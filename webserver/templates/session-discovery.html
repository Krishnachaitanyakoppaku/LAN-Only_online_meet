Find Meeting

Enter the session ID (server IP address) provided by the meeting host

The session ID is usually the host's IP address

How to get the Session ID?

  • Ask the host for their server IP address
  • It usually looks like: 192.168.1.100 or 10.0.0.5
  • The host can find it by running the server and checking the console
  • Or the host can share a direct link like: http://192.168.1.100:5000/simple-join

What happens next?

  • We'll check if the server is running at that IP address
  • If found, you'll be redirected to the meeting page
  • You can then enter your name and join the meeting
